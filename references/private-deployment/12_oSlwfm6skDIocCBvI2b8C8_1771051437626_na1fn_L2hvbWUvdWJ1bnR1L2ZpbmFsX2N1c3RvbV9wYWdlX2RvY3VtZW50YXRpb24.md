# 明道云私有部署：自定义页面模块深度解析

## 引言

明道云私有部署（HAP）平台中的**自定义页面**模块，是一个功能强大的应用构建工具，它赋予了应用管理员根据特定业务需求，创建高度定制化的信息展示与交互界面的能力。该模块通常被用于构建**仪表盘（Dashboard）、驾驶舱（Cockpit）或工作台（Workbench）**，旨在将来自不同数据源的关键业务信息进行聚合、可视化展示和深度分析，从而帮助用户快速洞察业务状况，做出精准的数据驱动决策 [1]。

本文档将深入剖析自定义页面模块，从核心概念、配置清单、操作步骤，到最佳实践、常见问题、模块关联及实际案例，为用户提供一份比官方文档更详尽、更易于理解的参考指南。

## 1. 核心概念与定义

自定义页面模块的核心价值在于其**信息聚合**与**高度可定制化**的能力。它解决了企业在数字化转型过程中，信息分散、数据孤岛、以及不同角色用户对信息需求各异的痛点。

### 1.1 解决的核心问题

| 核心问题 | 解决方案与价值 |
|---|---|
| **信息聚合与可视化** | 将来自不同工作表、统计图表、外部链接等多种来源的信息聚合在一个页面上，并通过丰富的组件进行可视化展示，帮助用户快速掌握业务概况。 |
| **个性化工作台** | 为不同角色或部门的用户提供定制化的工作界面，将他们最关心的信息和最高频的操作整合在一起，显著提高工作效率和数据分析的便捷性。 |
| **数据驱动决策** | 通过集成统计图表、筛选器等组件，支持用户进行实时数据分析和联动筛选，从而能够更深入地探索数据，发现业务规律，做出更明智的业务决策。 |
| **灵活的扩展性** | 支持嵌入外部URL、富文本等内容，使得自定义页面不仅限于展示平台内部数据，还能整合外部资源（如第三方BI系统、外部资讯网站等），实现更广泛的应用场景。 |

### 1.2 支持的核心组件

自定义页面的强大功能，是通过一系列可自由组合的组件来实现的。下表列出了其支持的核心组件及其功能：

| 组件 | 功能描述 |
|---|---|
| **统计图表** | 展示各种数据统计图，如饼图、柱状图、折线图、漏斗图等，是数据可视化的核心。 |
| **快捷按钮** | 用于触发特定操作或快速跳转到应用内的其他页面、工作表、视图，甚至外部链接。 |
| **筛选器** | 对页面上的数据进行全局或局部的筛选和过滤，支持用户按需查看数据。 |
| **轮播图** | 以轮播的形式展示多张图片或信息，常用于展示产品图片、公司新闻等。 |
| **视图** | 直接嵌入工作表的某个视图（如表格视图、看板视图），在自定义页面上展示原始数据或经过筛选的数据。 |
| **富文本** | 支持图文混排的富文本编辑器，用于在页面上展示说明性文字、公告、图片等内容。 |
| **嵌入的URL** | 将外部网页或应用无缝嵌入到自定义页面中，实现内部系统与外部资源的整合。 |

## 2. 完整配置清单

自定义页面的配置项主要分为**页面整体配置**和**组件配置**两大类，这些配置项共同决定了页面的外观、行为和功能 [1]。

### 2.1 页面整体配置

页面整体配置决定了自定义页面的基础风格和通用能力。

| 配置项 | 说明 | 可选值/示例 | 备注 |
|---|---|---|---|
| **页面背景色** | 设置自定义页面的背景颜色。 | 深色主题、浅色主题、固定的白灰黑三色 | 影响页面整体视觉风格。 |
| **背景图** | 选择页面背景图片。 | 6种预设图形可选 | 用于美化页面背景，增加视觉吸引力。 |
| **全局颜色** | 统一设置页面中所有统计图表的颜色。 | 自定义颜色 | 设置后将覆盖统计图表本身的颜色配置，但仍可在图表设置中为单个图表单独调整颜色。 |
| **链接参数** | 定义一个参数名，作为查询字符串附加在自定义页面链接后。 | 例如：`name` | 用于在加载页面时动态获取参数值，以筛选自定义页面中的筛选器和统计图表，实现页面的动态展示。 |
| **页面布局** | 设置页面中组件的排列方式。 | `滚动`：保留组件大小，页面高度不够时可滚动查看；`适应屏幕高度`：所有组件在一屏显示，可能会压缩显示。 | 根据组件数量和内容复杂度来决定，信息量大时建议使用“滚动”。 |
| **页面自动刷新** | 设置页面中统计图表或嵌入视图的自动刷新间隔。 | 最小30秒，最长5分钟 | 仅统计图表内容刷新，嵌入的视图和URL不会自动刷新。 |
| **联动筛选** | 开启后，相同数据源的统计图表之间支持联动。 | 开启/关闭 | 仅自定义页面中的统计图表支持联动，是实现深度数据分析的关键功能。 |
| **全屏查看** | 开启或关闭自定义页面的全屏查看功能。 | 开启/关闭 | 位于“页面配置”-“标题区”。 |
| **对外分享** | 开启或关闭自定义页面的对外分享功能。 | 开启/关闭 | 位于“页面配置”-“标题区”。不显示分享按钮表示分享功能关闭，已发出的链接也将失效。 |
| **保存为图片** | 开启或关闭将自定义页面保存为图片的功能。 | 开启/关闭 | 位于“页面配置”-“标题区”。 |

### 2.2 组件配置 (以统计图表为例)

每个组件都有其独立的配置项，以统计图表为例：

| 配置项 | 说明 | 可选值/示例 | 备注 |
|---|---|---|---|
| **数据表选择** | 选择统计图表的数据来源工作表。 | 例如：【销售线索】表 | 统计图表的基础数据来源。 |
| **图表类型选择** | 选择统计图表的展示类型。 | 饼图、柱状图、折线图等 | 根据数据特点和分析目的选择最合适的图表类型。 |
| **统计设置** | 进一步设置数据的统计方式、维度和聚合方式。 | 详细配置请参考[统计图表文档](https://help.mingdao.com/statistics/introduction) | 这是决定图表展示具体内容的核心配置。 |
| **自动刷新** | 设置单个统计图表的自动刷新间隔。 | 同页面自动刷新设置 | 继承页面整体设置，也可为重要图表单独设置更短的刷新间隔。 |
| **分享功能** | 开启或关闭单个统计图表的分享功能。 | 开启/关闭 | 位于“页面配置”-“图表操作”。 |
| **导出Excel功能** | 开启或关闭将单个统计图表的数据导出为Excel的功能。 | 开启/关闭 | 位于“页面配置”-“图表操作”。 |

## 3. 操作步骤详解

以下是从零开始创建一个功能完善的自定义页面的详细操作流程 [1]。

### 3.1 创建自定义页面

1.  **进入应用**：首先，登录明道云平台，并进入需要创建自定义页面的应用。
2.  **新建页面**：在应用界面，点击左侧导航栏的【新建】按钮，然后选择【创建自定义页面】。
3.  **选择画布类型**：在弹出的选项中，选择【画布】类型，这将为您提供一个空白的页面进行自由设计。
4.  **命名页面**：输入自定义页面的名称，例如“销售仪表盘”、“项目工作台”等，然后点击确认。

### 3.2 添加和配置组件

1.  **进入编辑模式**：页面创建后，会自动进入编辑模式。您也可以在页面列表中点击自定义页面名称进入。
2.  **添加组件**：在编辑界面的左侧，您会看到各种可用的组件列表。根据您的需求，将组件拖拽到页面画布中。
3.  **配置组件**：对拖入的每个组件进行详细配置。例如，对于统计图表，需要选择数据表、图表类型、设置统计维度等；对于快捷按钮，需要设置按钮名称和跳转链接。
4.  **调整组件大小和位置**：通过拖拽组件的边框或右下角，可以调整组件的大小。点击组件顶部空白区域并拖拽，可以调整组件在页面上的位置。

### 3.3 页面整体配置

1.  **进入页面配置**：在自定义页面编辑模式下，点击页面右上角的【页面配置】按钮。
2.  **进行配置**：根据前文“2.1 页面整体配置”表格中的说明，对页面背景、全局颜色、链接参数、页面布局、自动刷新、联动筛选等进行设置。

### 3.4 APP端显示效果设置

1.  **进入APP端设置**：在自定义页面编辑模式下，通常会有针对APP端显示效果的独立设置入口。
2.  **隐藏或调整组件**：根据APP端的显示需求，可以隐藏不需要在APP端显示的组件，或单独调整组件的顺序，以优化移动端用户的体验。

### 3.5 页面分享与导出

1.  **配置分享与导出**：在“页面配置”-“标题区”和“图表操作”中，开启或关闭全屏查看、对外分享、保存为图片、导出Excel等功能。
2.  **选择分享方式**：如果开启了对外分享，可以选择以下两种方式：
    *   **嵌入链接方式**：通过链接将自定义页面嵌入到其他系统，需要用户登录并根据权限查看。
    *   **公开分享方式**：将自定义页面内容公开，无需登录即可查看。可设置页面标题、有效期和密码。此功能需应用管理员开启。

## 4. 最佳实践

遵循以下最佳实践，可以帮助您创建出更高效、更美观、更实用的自定义页面。

*   **目标导向设计**：在创建自定义页面之前，务必明确该页面的核心目标和主要用户群体。这将有助于您选择最合适的组件和布局，避免信息冗余。
*   **布局与简洁性**：对于信息量大的页面，建议选择“滚动”布局，确保所有组件都能以完整大小显示。对于需要一览无余的概览页面，可以选择“适应屏幕高度”布局。同时，保持页面整洁，突出重点，避免在单个页面上堆砌过多的组件。
*   **善用交互功能**：
    *   **巧用链接参数**：通过设置链接参数，实现自定义页面的动态筛选，极大地提升页面的复用性和灵活性。
    *   **开启联动筛选**：对于包含多个相关统计图表的页面，务必开启联动筛选功能，以实现深度的数据探索和分析。
*   **移动端优先**：在当今移动办公的趋势下，务必利用APP端显示效果设置功能，对移动端页面进行单独优化，确保移动端用户也能获得良好的体验。
*   **安全分享**：在分享页面时，根据信息敏感度和分享对象，谨慎选择“嵌入链接”或“公开分享”方式，并为公开分享设置必要的密码和有效期，确保数据安全。

## 5. 常见问题与解决方案

| 常见问题 | 解决方案 |
|---|---|
| **自定义页面中统计图表数量限制** | 目前自定义页面最多支持设置30个统计图表。如果需要展示更多图表，建议将图表进行分组，或创建多个自定义页面。 |
| **页面自动刷新范围** | 页面自动刷新仅支持统计图表内容的刷新。嵌入的视图、URL等组件不会自动刷新，需要手动点击浏览器刷新按钮才能更新内容。 |
| **统计图表联动筛选不生效** | 请检查：1. 数据源是否相同（联动筛选仅支持相同数据源的图表）；2. 是否在“页面配置”中开启了“同数据源组件自动关联”。 |
| **页面分享链接失效或不显示分享按钮** | 请检查：1. “页面配置”-“标题区”中“对外分享”功能是否已开启；2. 公开分享功能是否由应用管理员开启；3. 分享链接是否在有效期内，密码是否正确。 |
| **APP端显示效果不佳** | 请进入自定义页面的APP端设置，单独为APP调整显示效果，例如隐藏不必要的组件，或调整组件之间的顺序。 |
| **嵌入链接方式查看页面需要登录** | 这是正常现象。嵌入链接方式默认需要用户登录并根据权限查看。如果希望无需登录即可查看，请使用“公开分享方式”。 |

## 6. 与其他模块的关联

自定义页面模块与明道云平台的多个核心模块紧密协作，共同构建起强大的应用功能。

*   **与工作表、统计图表、视图的关联**：工作表是数据的基础，统计图表和视图是数据的呈现方式，而自定义页面则是这些数据呈现方式的聚合与展示平台，三者共同构成了从数据存储到数据可视化的完整链路。
*   **与外部系统/URL的关联**：通过嵌入URL组件，自定义页面能够无缝集成外部系统，实现内外信息的整合。同时，自定义页面本身也可以被分享或嵌入到外部系统中，实现内外互通。
*   **与权限管理的关联**：自定义页面的访问和操作权限，都受到平台权限管理模块的严格控制，确保了数据的安全与合规。

## 7. 实际案例

### 7.1 销售团队业绩仪表盘

**场景**：销售总监需要实时监控团队业绩，包括销售额、订单量、客户转化率等，并能按区域、产品线、销售人员等维度进行筛选和分析。

**解决方案**：创建一个名为“销售团队业绩仪表盘”的自定义页面，其中包含销售额趋势图、订单量分布图、客户转化率漏斗图、销售人员业绩排行榜等多个统计图表，并配置时间、区域、产品线、销售人员等多个筛选器。同时，开启图表间的联动筛选，并设置页面每5分钟自动刷新一次。通过该仪表盘，销售总监可以清晰地看到公司整体销售概况，并通过筛选器和联动功能快速定位问题、调整策略。

### 7.2 项目管理工作台

**场景**：项目经理需要一个集中的工作台，能够概览所有项目的进度、任务分配、风险预警等信息，并能快速进入具体项目进行管理。

**解决方案**：创建一个名为“项目管理工作台”的自定义页面，其中包含项目进度概览图、任务完成情况图、团队成员工作负荷图等统计图表，并嵌入“待办任务列表”和“项目风险列表”两个视图。同时，配置“新建项目”、“查看所有任务”等快捷按钮，并嵌入项目管理工具（如Jira）的看板链接。通过该工作台，项目经理可以快速了解所有项目的健康状况，及时发现潜在风险，并合理分配团队资源，大大提高了项目管理的效率。
